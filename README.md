# RoomMaker â€“ Revit Design Automation Plugin

This project builds a simple Revit Design Automation (DA) plugin that:

- Reads a JSON file with basic room measurements.
- Creates a rectangular room (walls + floor) in a Revit model.

It targets:

- .NET Framework 4.8 (`net48`)
- Autodesk APS Design Automation for Revit

The DLL and bundle generated by this repo are meant to be used as an AppBundle
for APS Design Automation.

## Built artifacts

GitHub Actions builds this project on Windows and uploads:

- `RoomMaker.bundle.zip`

The ZIP contains:

- `RoomMaker.bundle/Contents/RoomMaker.dll`
- `RoomMaker.bundle/Contents/RoomMaker.addin`
- `RoomMaker.bundle/PackageContents.xml`

You can download this artifact from the **Actions** tab after a successful run.
